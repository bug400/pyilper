User Interface

Before you launch pyILPER connect the PC to the PIL-Box and set up the HP-IL loop with at least the HP-IL controller (HP41 or HP-71B). After the launch of pyILPER you must reconfigure the HP-IL devices on the loop, by either powering the controller off and on again or by issuing a RESTORE IO on the HP-71B.

The user interface consists of:

  • The file menu to configure pyILPER
  • The utilities menu
  • The help menu
  • Tabbed virtual HP-IL devices: scope, printer(s), drive(s), raw drive(s) , terminal, HP7470A plotter, HP82162A thermal printer, HP2225B ThinkJet printer
  • Status information at the bottom of the main window

The tabbed virtual HP-IL devices can be undocked and moved to floating, resizeable windows by pressing the undock button at the right corner of the tab bar. Clicking the close button of a floating window docks the virtual device into the tab again.

The pyILPERmain window with the tabbed devices or all undocked device windows can be resized to an appropriate size. The number of lines of scope, terminal, printer or plotter output will increase or decrease accordingingly.
