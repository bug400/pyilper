Release Notes for pyILPER 1.8.0

  • Added the HP-2225B ThinkJet printer.
  • Added virtual terminal support for the HP-75
  • Various dokumentation amendments
  • The minimum required version for Python is now 3.5 instead of 3.4. Features of Python 3.5 and above make the code for calling external applications more clearly and easy to maintain.

Potential incompatibility:

Due to the introduction of the HP-75 character set the configuration of character sets in existing pyILPEER installations must be ajdusted if the Roman-8 character set was used.
